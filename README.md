## Introduction

## Important

Edit go.mod and rename to your package module

## Commit rules
Commit message should looks like
```
[TYPE] some message
```

### Types
 - Add
 - Fix
 - Update
 - Remove
 - Refactor
 - Docs
 - Test
 - Improve
